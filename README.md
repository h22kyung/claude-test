# VOC Excel 정리 자동화

매일 다운로드하는 VOC Excel 파일을 자동으로 정리하는 Python 스크립트입니다.

## 하는 일

1. **필터링** — `세부시스템`이 지정값(a,b,c)이고, `답변자이름`이 지정값(가,나,다,라)이며,
   `row상태`가 `접수`가 아닌 행만 남깁니다.
2. **불필요 행 제거** — `tat(hr)`이 0이면서 `row상태`가 `이관`인 행을 제거합니다.
3. **컬럼 선별·재정렬** — 아래 12개 컬럼만 이 순서로 남기고 나머지는 모두 삭제합니다.
   > voc id, voc유형, 제목, row상태, voc등록자, voc등록자id, voc등록자부서,
   > voc등록일, 현재상태, 처리결과, 답변자이름, 답변일
4. **소요시간 매핑** — `가/나/다/라` 4개 컬럼을 추가하고, 같은 워크북의 소요시간 시트
   (voc id, 소요시간, 답변자이름)를 참고해 각 voc id의 소요시간을 해당 답변자 컬럼에
   기록합니다. (voc id당 답변자 1명 기준, 나머지 3개 컬럼은 공란)
   소요시간 시트에 없는 voc id는 기본값 **0.5**로 채웁니다.

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python voc_cleaner.py 입력파일.xlsx
```

- 출력 파일명을 직접 지정하려면:
  ```bash
  python voc_cleaner.py 입력파일.xlsx 결과.xlsx
  ```
- 지정하지 않으면 `입력파일_정리_YYYYMMDD.xlsx` 형식으로 같은 폴더에 생성됩니다.

## 설정 변경 (중요)

`voc_cleaner.py` 파일 상단의 **CONFIG** 영역만 수정하면 됩니다.
a, b, c 와 가, 나, 다, 라 는 예시값이므로 **실제 세부시스템명/답변자이름으로 교체**하세요.

```python
SUBSYSTEM_FILTER = ["a", "b", "c"]            # 세부시스템: 이 값만 남김
RESPONDER_FILTER = ["가", "나", "다", "라"]    # 답변자이름: 이 값만 남김 (= 추가 컬럼명)
MAIN_SHEET_NAME = 0                            # 메인 데이터 시트 (시트명 또는 0=첫 시트)
TIME_SHEET_NAME = "소요시간"                    # 소요시간 표가 있는 시트명
DEFAULT_DURATION = 0.5                         # 소요시간 시트에 없는 voc id의 기본값
```

- `RESPONDER_FILTER`를 바꾸면 추가되는 컬럼명(가/나/다/라)도 자동으로 같이 바뀝니다.
- 컬럼명은 공백·대소문자 차이를 무시하고 자동으로 찾으므로
  (`voc등록자id` ↔ `voc등록자 id` 등) 약간의 표기 차이는 괜찮습니다.

## 시트 이름에 대하여

시트 이름이 `Sheet1`, `Sheet2`처럼 일반적이어도 동작합니다.

- **데이터 시트**: 첫 번째 시트를 사용하므로 이름은 무관합니다.
  (데이터가 첫 시트가 아니면 `MAIN_SHEET_NAME`에 실제 시트명을 입력하세요.)
- **소요시간 시트**: 다음 순서로 자동으로 찾습니다.
  1. `TIME_SHEET_NAME`과 이름이 일치하는 시트
  2. (없으면) `소요시간`·`voc id` 컬럼을 가진 시트를 자동 감지 — 시트 순서 무관
  3. (그래도 없으면) 두 번째 시트

## 참고

- 소요시간 시트에 없는 voc id는 해당 컬럼이 기본값 `0.5`로 채워집니다.
- 필요한 컬럼/시트를 찾지 못하면 한글 안내 메시지와 함께 실제 컬럼/시트 목록을 출력합니다.
